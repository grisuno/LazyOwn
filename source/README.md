# source/

Sphinx documentation source. Drives the developer-facing API docs that
ship alongside the auto-generated `README.md` / `docs/index.html`. The
operator-facing docs live in `docs/` and are regenerated by `DEPLOY.sh`.

## Files

| File | Purpose |
|------|---------|
| `conf.py` | Sphinx configuration (project name, theme, extensions). |
| `index.rst` | Root reStructuredText entry point. Lists modules and includes. |

## Building locally

```sh
pip install sphinx
sphinx-build -b html source/ build/sphinx/
```

The build is informational — the canonical operator docs are
auto-regenerated by `DEPLOY.sh` from `UTILS.md` + `COMMANDS.md` +
`CHANGELOG.md`.
